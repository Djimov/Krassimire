# Fluxos de utilização

## Fluxo principal
1. Utilizador abre a aplicação.
2. Define uma região no mapa.
3. Introduz intervalo temporal.
4. Define cobertura máxima de nuvens.
5. Executa pesquisa.
6. Consulta resultados na timeline.
7. Selecciona uma imagem para ver detalhe.
8. Selecciona uma segunda imagem para comparação.

## Fluxos alternativos
- datas inválidas;
- ausência de resultados;
- falha de comunicação com a API externa;
- credenciais OAuth ausentes para Process API.
