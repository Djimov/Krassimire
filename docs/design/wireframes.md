# Wireframes e desenho inicial

## Objectivo
Este documento descreve a estrutura visual prevista para o MVP antes da implementação detalhada da interface.

## Ecrã principal
O ecrã principal deverá conter:

- cabeçalho com título do projecto;
- área central com mapa interactivo;
- painel lateral ou superior com filtros de pesquisa;
- botão para iniciar pesquisa;
- zona inferior ou lateral para timeline de resultados.

## Ecrã de visualização individual
Após selecção de um resultado, a interface deverá apresentar:

- a imagem escolhida;
- informação temporal básica;
- selector de composição de bandas;
- opção de envio da imagem para comparação.

## Ecrã de comparação temporal
A vista de comparação deverá incluir:

- duas imagens lado a lado;
- identificação temporal de cada uma;
- indicação do modo de visualização utilizado;
- possibilidade de regressar à timeline.
